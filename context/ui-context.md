## UI Context

### Theme
- **Mode:** Light (default), dark as preference.
- **Background:** `bg-gray-50` light, `bg-gray-900` dark.
- **Border:** `border-gray-200` light, `border-gray-700` dark.

### Colors
- **Primary:** `#2563eb` (blue-600).
- **Secondary:** `#8b5cf6` (violet-600).
- **Success:** `#10b981` (emerald-500).
- **Danger:** `#ef4444` (red-500).
- **Neutral text:** `#374151` (gray-700) light, `#d1d5db` (gray-300) dark.

### Typography
- **Headings:** Inter bold.
- **Body:** Inter regular.
- **Code:** JetBrains Mono.

### Component Conventions
- Use Tailwind CSS utility classes.
- Reusable components in `src/components/ui/`.
- All interactive elements have focus-visible ring.
